# Czerepaha

A Jekyll site using the Minima theme.

## Setup

1. Install Ruby and Bundler if you haven't already
2. Run `bundle install` to install dependencies
3. Run `bundle exec jekyll serve` to start the development server
4. Visit `http://localhost:4000` in your browser

## Deployment

This site is ready to be deployed to GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in your repository settings.

## Customization

- Edit `_config.yml` to change site settings
- Add new posts in the `_posts` directory
- Modify `index.md` and `about.md` to customize pages
- See [Jekyll documentation](https://jekyllrb.com/docs/) for more options
